# Departmental Store Management System

## Instructions:
1. Copy the 'departmental_store' folder into XAMPP's `htdocs` directory.
2. Start Apache and MySQL from XAMPP Control Panel.
3. Import `store_db.sql` into phpMyAdmin.
4. Open `http://localhost/departmental_store/login.php` in your browser.
